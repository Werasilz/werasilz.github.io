About me

             นายวีรศิลป์ ฤกษ์เวียง เป็นโปรแกรมเมอร์สายพัฒนาเกม มีความชื่นชอบในการเล่นเกมและสร้างเกมของตัวเอง เริ่มต้นเรียนรู้การสร้างเกมตั้งแต่ ม.ต้น จนจบ ม.ปลาย ได้มีผลงานการส่งแข่งขันมากมาย ปัจจุบันศึกษาสาขาวิชาด้านการสร้างเกมโดยเฉพาะ เริ่มเขียน C# ตั้งแต่ ปี 1 ปัจจุบันมีผลงานเกมหลายอย่าง และจะเห็นได้ถึงพัฒนาการในการสร้างเกม ที่เริ่มต้นจาก No coding ไปจนถึงเกม Virtual Reality

RSU

     ศึกษาที่ สาขาคอมพิวเตอร์เกมมัลติมีเดีย วิทยาลัยนวัตกรรมดิจิทัลเทคโนโลยี มหาวิทยาลัยรังสิต

Unity

     มีประสบการณ์การใช้งานโปรแกรม Unity 2 ปี มีผลงานสร้างเกมมากมายเช่น 2D 3D AR VR

C#

     มีความรู้และประสบการณ์การเขียนโปรแกรมด้วยภาษา C# เป็นหลัก โดยใช้ในการสร้างเกมในโปรแกรม Unity

Portfolio

Ducky Bomb

เกม 2D endless run ออกแบบมาในแนวตั้งเพื่อเล่นในโทรศัพท์มือถือ บังคับเกมได้ด้วยการกดจอด้านซ้ายและขวา

Earth Protector

เกม 3D AR สร้างโดยใช้ Unity AR foundation ทำงานโดยใช้ Plane Dectection และควบคุมโดย Device

TukTuk 2077 Express

2 Player 2D Game ธีม Thailand 2077 แข่งขันกัน 2 คน รับ-ส่งของให้เมืองต่าง ๆ สะสมแต้มและโบนัส

Zombie Shooter AR

เกม AR ยิงซอมบี้ใช้ระบบการยิงจากตัวกล้องใส่ซอมบี้ ซอมบี้เกิดตามจุดพิกัดใน GPS และจะเดินหาผู้เล่น

Adventure 2D

เกม 2D RPG แนวตั้งเหมาะสำหรับการเล่นบนมือถือ มีระบบต่าง ๆ มากมาย มีศัตรูหลายแบบ

Adventure Demon World AR

ผลงานส่งโครงการ NSC 2021 เกม RPG AR โดยใช้กราฟิกแบบ 2D มีระบบพื้นฐานความเป็น RPG และด่านมากมายให้เล่น

Annoying Party House

เกมที่พัฒนาในงาน Global Game Jame From Home 2020 เกม 3D ทำภารกิจต่าง ๆ ในบ้านที่จัดปาร์ตี้

VR Shooting Game

ผลงานเกม VR เกมแรกแนว Action-Shooting กำจัดศัตรูและช่วยเหลือผู้เล่น พร้อมลูกเล่นที่แปลกใหม่และท้าทาย
